# 📝 Go-Todo CLI
A simple Command Line Interface (CLI) Todo List application written in Go.

## 🚀 Features
Create: Add new tasks to your list.

Read: View all your tasks with their status (Done/Pending).

Update: Mark tasks as completed or edit task descriptions.

Delete: Remove tasks from the list by ID.

Persistence: Data is saved locally (JSON/File) so you don't lose your tasks.

## 🛠️ Tech Stack
Language: Go (Golang)

Data Format: JSON / Plain Text

Packages: os, fmt, encoding/json
